# Communications Director Application | Jones County School System

Portfolio website for Katherine Scarlett Scoville's application for the Communications Director position at Jones County School System.

**View the live site:** [scarlettscoville.github.io/jccommunicationdirector](https://scarlettscoville.github.io/jccommunicationdirector)

## What's Here

- **Performance Task** — District narrative, campaign plan, and 90-Day Momentum Strategy
- **Media Campaign Gallery** — Social media graphics and caption copy for the Jones County Greyhounds campaign, including a launch post, 3-day countdown series, and game day post
- **Launch Emails** — Internal and external email communications drafted as part of the campaign strategy
- **Microsoft Teams Structure** — A proposed two-tier internal communications architecture for the district
